# Swift Exercism Solutions

Este repositorio contiene mis soluciones a los ejercicios de **Exercism**, implementados en el lenguaje **Swift** y validados previamente en la plataforma.


---

## 📚 Sobre Exercism

**Exercism** es una plataforma que ofrece ejercicios de programación con distintos niveles de dificultad, diseñados para mejorar habilidades prácticas mediante la resolución de problemas y retroalimentación automatizada.

---

## 🧩 Contenido del repositorio

* Ejercicios de diferentes niveles:

  * 🟢 Básico
  * 🟡 Intermedio
  * 🔴 Avanzado

* Cada solución:

  * Está escrita en **Swift**
  * Ha sido **validada en Exercism**
  * Representa una implementación funcional del problema planteado

---

## 🧠 Metodología de trabajo

El flujo seguido para cada ejercicio es:

1. Resolver el problema localmente en Swift
2. Enviar la solución a Exercism para validación
3. Ajustar el código hasta que cumpla con los requisitos
4. Subir la solución final a este repositorio

Este proceso permite reforzar tanto la lógica de programación como el uso práctico de herramientas de desarrollo.

---

## 🎯 Objetivos

El objetivo no es solo resolver problemas, sino también practicar un flujo de trabajo más cercano al desarrollo real: escribir código, validarlo externamente y gestionarlo mediante control de versiones.

---

## 🚀 Progreso

El repositorio se actualiza constantemente con nuevos ejercicios conforme se avanza en la plataforma.
